DLive Sticker Quick 隱私權政策

最後更新日期：2026年3月25日

簡單來說：我們不會收集、不會儲存、無法存取您的任何資料。所有資料都只保存在您自己的裝置上。

1. 資料收集與使用

DLive Sticker Quick 擴充功能僅在您本地裝置上儲存以下資料：

  • 您添加的貼圖 ID 清單
  • 貼圖的標籤與分類資訊
  • 您標記的最愛貼圖
  • 自訂的標籤詞彙

這些資料使用 Chrome 瀏覽器的 chrome.storage.local API 儲存，完全保存在您的裝置上。

2. 資料不上傳

我們不會：

  • 將您的資料上傳到任何外部伺服器
  • 將資料傳輸給第三方
  • 追蹤您的使用行為或瀏覽紀錄
  • 分析或販售您的資料

3. 開發者無法存取

由於資料僅儲存在您的本地裝置，開發者完全無法存取您的任何貼圖資料或設定。

4. 聯絡方式

如有任何隱私權相關問題，請透過 Chrome Web Store 的開發者資訊頁面與我們聯繫。
