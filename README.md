## 📦 Installation

1. Download ZIP (Click "Code" → "Download ZIP")
2. Unzip the file
3. Open Chrome and go to `chrome://extensions/`
4. Enable "Developer mode"
5. Click "Load unpacked" and select the unzipped folder
6. Done! You'll see the 🎨 icon in your toolbar

## 📖 How to Use

1. Go to any DLive channel (login first)
2. You'll see a 🎨 button in the chat area
3. Click to open the sticker panel and send stickers

### Manage Stickers
- Click the 🎨 icon in toolbar to open manager
- Paste sticker IDs (one per line)
- Click "★" to favorite
- Click "✕" to delete

### Right-click Quick Add
- Right-click on any sticker → Add to Sticker Quick

<a name="中文"></a>
## 📦 安裝方法

1. 下載 ZIP（點右上角「Code」→「Download ZIP」）
2. 解壓縮
3. 打開 Chrome 輸入 `chrome://extensions/`
4. 開啟「開發者模式」
5. 點「載入未封裝項目」，選擇解壓縮的資料夾
6. 完成！工具列會出現 🎨 圖示

## 📖 怎麼用

1. 去 DLive 任一頻道（要先登入）
2. 聊天室會多一個 🎨 按鈕
3. 點開就可以選貼圖送出

### 管理貼圖
- 點工具列 🎨 打開管理介面
- 貼上貼圖 ID（一行一個）
- 點「★」設常用
- 點「✕」刪除

### 右鍵快速新增
- 對貼圖按右鍵 → 新增到 Sticker Quick
