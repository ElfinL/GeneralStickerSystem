General Sticker System (GSS)

通用貼圖系統

📋 快速開始

使用步驟：

  1. 在 DLive 或 Twitch 登入並進入頻道頁面
  2. 開啟插件，輸入貼圖 ID 清單
  3. 點擊聊天室旁的 GSS 圖示開啟貼圖面板
  4. 一鍵發送貼圖！

🎨 貼圖格式

DLive 官方貼圖

格式：DL-xxx

  • 在 DLive 頻道頁面右鍵點擊貼圖即可快速儲存 ID
  • 或在貼圖連結中找到 ID（例如：DL-826c4ac1e004273_498281）

🔍 如何從網址取得 ID：

貼圖網址範例：https://images.prd.dlivecdn.com/emote/826c4ac1e004273_498281

網址最後一段 826c4ac1e004273_498281 就是 ID
使用時請加上 DL- 前綴，變成 DL-826c4ac1e004273_498281

Imgur 圖片/影片

格式：IM-xxx.ext

  • 支援格式：.gif, .png, .jpg, .mp4
  • 範例：IM-ha3eTC7.gif、IM-abc1234.mp4

🔍 Imgur ID 取得方式：

Imgur 網址範例：https://i.imgur.com/ha3eTC7.gif 或 https://imgur.com/ha3eTC7

網址最後一段 ha3eTC7 就是 ID
使用時請加上 IM- 前綴和副檔名（如 .gif/.mp4），變成 IM-ha3eTC7.gif
※ 若不知道副檔名，可先嘗試 .gif 或 .mp4

meee.com.tw 圖片/影片

格式：ME-xxx.ext

  • 支援格式：.gif, .jpg（其他格式待確認）
  • 範例：ME-LXEcxGi.gif、ME-abc1234.jpg

🔍 meee ID 取得方式：

meee 網址範例：https://i.meee.com.tw/LXEcxGi.gif

網址最後一段 LXEcxGi 就是 ID
使用時請加上 ME- 前綴和副檔名（如 .gif/.jpg），變成 ME-LXEcxGi.gif
※ 若不知道副檔名，可先嘗試 .gif 或 .jpg

🏷️ 標籤功能

每張貼圖可添加最多 4 個標籤，方便分類和搜尋。

格式：DL-xxx #meme #reaction

  • 標籤以 # 開頭
  • 每個標籤最多 16 個字元
  • 不能包含空格或 #

💡 提示：右鍵點擊貼圖（在頻道頁面或插件面板）即可快速添加/移除標籤

⭐ 快捷操作

  • ★ 常用標記：點擊貼圖上的 ★ 按鈕將其標記為常用，常用貼圖會置頂顯示
  • ✕ 刪除：點擊 ✕ 按鈕從清單中移除貼圖
  • 右鍵標籤：在貼圖上點擊右鍵可添加或移除標籤

🎭 DLive 劇院模式

DLive 設定頁提供多種視覺調整功能：

  • 劇院模式：全屏觀看體驗
  • 聊天室控制：調整聊天室寬度或隱藏
  • 元素控制：隱藏導航欄、標題、送禮區等

🎮 全屏模式快捷鍵：

  • 按 C 鍵：快速開關聊天室
  • 滑鼠移到畫面右側：自動展開聊天室

※ 快捷鍵僅在全屏模式下有效，劇院模式不適用

📋 Quick Start

Steps:

  1. Log in to DLive or Twitch and go to a channel page
  2. Open the extension and enter your sticker ID list
  3. Click the GSS icon next to the chat to open the sticker panel
  4. Send stickers with one click!

DLive Official Emotes

Format: DL-xxx

  • Or find the ID in the emote URL (e.g., DL-826c4ac1e004273_498281)

Example URL: https://images.prd.dlivecdn.com/emote/826c4ac1e004273_498281

🏷️ Tags

Each sticker can have up to 4 tags for easy categorization and filtering.

Format: DL-xxx #meme #reaction

  • Tags start with #
  • Max 16 characters per tag
  • No spaces or # allowed in tag names

💡 Tip: Right-click on any sticker (on channel or in panel) to quickly add/remove tags

⭐ Shortcuts

  • ★ Favorite: Click the ★ button to mark a sticker as favorite. Favorites appear at the top.
  • ✕ Delete: Click the ✕ button to remove a sticker from your list
  • Right-click Tags: Right-click a sticker to add or remove tags

📋 クイックスタート

使用手順：

  1. DLive または Twitch にログインしてチャンネルページへ
  2. 拡張機能を開き、ステッカーIDリストを入力
  3. チャット横の GSS アイコンをクリックしてパネルを開く
  4. ワンクリックでステッカーを送信！

🎭 DLive シアターモード

DLive 設定ページでは、さまざまな視覚調整機能を提供しています：

  • シアターモード：全画面表示体験
  • チャット制御：チャット幅の調整または非表示
  • 要素制御：ナビバー、タイトル、ギフトエリアなどを非表示

🎮 全画面ショートカット：

  • C キー：チャットのオン/オフ切り替え
  • マウスを右端に移動：チャットを自動展開

※ ショートカットは全画面モードでのみ有効。シアターモードでは無効

📋 빠른 시작

사용 방법:

  1. DLive 또는 Twitch에 로그인하고 채널 페이지로 이동
  2. 확장 프로그램을 열고 스티커 ID 목록을 입력
  3. 채팅창 옆 GSS 아이콘을 클릭하여 스티커 패널 열기
  4. 원클릭으로 스티커 보내기!

  • 지원 형식: .gif, .png, .jpg, .mp4

🔍 Imgur ID 가져오기:

※ 확장자를 모르면 .gif 또는 .mp4 시도

🏷️ 태그 기능

스티커당 최대 4개 태그를 추가하여 분류 및 검색이 쉬워집니다.

형식: DL-xxx #meme #reaction

  • 태그는 # 으로 시작
  • 태그당 최대 16자까지
  • 태그명에 공백이나 # 사용 불가

💡 팁: 스티커를 우클릭(채널 페이지 또는 패널 내)하여 태그를 빠르게 추가/제거

⭐ 단축키

  • ★ 즐겨찾기: ★ 버튼을 클릭하여 스티커를 즐겨찾기에 추가. 즐겨찾기는 상단에 표시됨
  • ✕ 삭제: ✕ 버튼을 클릭하여 목록에서 스티커 제거
  • 우클릭 태그: 스티커를 우클릭하여 태그 추가/제거
